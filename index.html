A Personal Access Token lets this page commit changes directly to your GitHub repo. Your token is stored only in your browser's localStorage — never sent anywhere except the GitHub API.

  2. Name it anything  |  Expiration: No expiration
  3. Check repo scope only
  4. Click Generate token — copy it immediately
